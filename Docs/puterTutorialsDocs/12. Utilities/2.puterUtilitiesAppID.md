# puter.appID
* * *

A property of the `puter` object that returns the App ID of the running application.

[](#syntax)Syntax
-----------------

```
puter.appID

```


[](#examples)Examples
---------------------

**Get the ID of the current application**

```
<html>
<body>
    <script src="https://js.puter.com/v2/"></script>
    <script>
        puter.print("App ID: " + puter.appID);
    </script>
</body>
</html>

```
